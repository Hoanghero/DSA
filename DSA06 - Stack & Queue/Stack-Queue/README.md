# Stack-Queue

Data Structures Implementation in C  
This repository contains comprehensive implementations of Stack and Queue data structures in C using various methods and approaches to demonstrate a deep understanding of their mechanics and applications.

## Features

### Stack Implementations

- Using Arrays (static size): `Stack/stackUsingArray.c`
- Using Linked Lists (dynamic size): `Stack/stackUsingLinkedList.c`
- Stack implemented using Two Queues (advanced): `Stack/StackUsingQueue.c`
- Abstract Data Type (ADT) for Stack: `Stack/stackUsingADT.c`

### Queue Implementations

- Using Arrays (circular and linear): `Queue/queueUsingArray.c`
- Using Linked Lists: `Queue/queueUsingLinkedList.c`
- Queue implemented using Two Stacks: `Queue/QueueUsingStack.c`
- Abstract Data Type (ADT) for Queue: `Queue/QueueUsingADT.c`

## Why this project?

Data structures like stacks and queues are fundamental building blocks in computer science. Implementing them from scratch in C helps you understand memory management, pointer manipulation, and algorithmic logic. Additionally, advanced implementations (e.g., stack using two queues) showcase problem-solving skills beyond the basics.

## How to use?

Each implementation is provided as a standalone C file with a `main` function demonstrating basic operations like push, pop, enqueue, dequeue, and display. Simply compile and run the files using:

```cmd
gcc filename.c -o output.exe
output.exe
```

## Contents

- `Stack/stackUsingArray.c` — Stack using arrays
- `Stack/stackUsingLinkedList.c` — Stack using linked list
- `Stack/StackUsingQueue.c` — Stack using two queues
- `Stack/stackUsingADT.c` — Stack as Abstract Data Type
- `Queue/queueUsingArray.c` — Queue using arrays
- `Queue/queueUsingLinkedList.c` — Queue using linked list
- `Queue/QueueUsingStack.c` — Queue using two stacks
- `Queue/QueueUsingADT.c` — Queue as Abstract Data Type
